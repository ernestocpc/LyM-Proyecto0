NOTA IMPORTANTE: El condicional 'not' debe ir con espacio despues. 
Ej: 
  (if (not (facing-p :north)) (put :balloons 1) (put :balloons 1)) -> Valido
  (if (not(facing-p :north)) (put :balloons 1) (put :balloons 1)) --> Invalido.
